Obsidian Chat Bridge

Adds bridge controls to supported chat UIs and can load current Obsidian context.

Local REST API endpoint: https://127.0.0.1:27124

Setup

  1. Install and enable the Obsidian Local REST API plugin in Obsidian.
  2. Copy the Local REST API key into the field below.
  3. Enter your vault name exactly as Obsidian uses it.
  4. If HTTPS requests fail, download the local certificate and install it into the Root Certificate Authority store.

Download Local REST API certificate

After changing settings or installing the certificate into the Root Certificate Authority store, save here and reload the extension if needed.
